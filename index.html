🎯 Objetivo del Portafolio

El presente portafolio de evidencias tiene como propósito documentar y demostrar el aprendizaje adquirido a lo largo del curso de Nuevas Tecnologías. Reúne las prácticas desarrolladas durante el ciclo escolar, organizadas de forma progresiva, desde el manejo básico de HTML y CSS hasta la implementación de aplicaciones web full-stack con arquitectura MVC, Express.js y MySQL.


Cada práctica refleja la aplicación de conceptos vistos en clase y el desarrollo de habilidades en programación web, lógica algorítmica, diseño de interfaces y construcción de sistemas de gestión de datos.

📖 Elementos Aprendidos

Estructura HTML semántica CSS y diseño responsive Etiquetas de sección (nav, header, footer, aside) Formularios y validación Métodos GET y POST JavaScript vanilla (DOM) Algoritmia y lógica de programación Funciones y modularidad en JS Manejo de eventos Arquitectura Cliente–Servidor Node.js y Express.js API REST (CRUD) MySQL y consultas SQL Arquitectura MVC CORS y comunicación frontend–backend Control de versiones con Git y GitHub
