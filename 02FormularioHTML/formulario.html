Todos los formularios que estan en internet tienen metodos de programacion por defecto. Estos metodos nos riven para hacer las acciones principalos son: Acciones para poderse conectar con los servidores a traves de las arquitecturas cliente servidor. Es que vamos a tener por defecto que todos los formularios utilizan el metodo GET, y el segundo metodo principal es POST, aunque existen otros metodos que se encargan del intercambio de información. GET; Este metodo envia por medio de la URL todo la informacion del formulario a traves del navegador. Su funcion principal es obtener los datos del servicio,pero es el metodo ms vulnerable de todos derivado a que se pueden generar inyecciones de codigo. POST: Este metodo se encarga de registrar una nueva entradaal sistema tambien utuliza la URL como envio por defecto pero su facultad es poder ocultarla., esto significa que los datos no son visibles para el usuario comun en el navegador. (PUT es para actualizar, DELETE para eso)
